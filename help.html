How to Get Your Gemini API Key

Follow these simple steps to activate Kenowa AI with the free Gemini API.

1
Go to Google AI Studio

Click the button below to visit Google AI Studio. You may need to sign in with your Google account.

Open Google AI Studio
2
Create API Key

Once logged in, click on the blue "Create API key" button.

If prompted, select "Create API key in new project" or select an existing project.

3
Copy Your Key

Your API key will be generated. Click the Copy icon next to the key.

Note: It usually starts with AIza...

4
Paste in Kenowa AI

Go back to the Kenowa AI extension and paste the key into the API Key field.
